# Telegram Bot

Простой Telegram бот, написанный на Python с использованием библиотеки python-telegram-bot.

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Получите токен бота от [@BotFather](https://t.me/botfather) в Telegram.

3. Вставьте токен в файл `.env`:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_здесь
   ```

4. Запустите бота:
   ```
   python main.py
   ```

## Команды

- `/start` - Приветственное сообщение
- `/help` - Информация о возможностях бота
